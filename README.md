# 🧬 BioRender Infographic Protocol  
**Assignment 2 – Abstract / Infographics**

---

## 🎯 Objective
To convert a scientific research protocol into a clear, visually structured infographic using BioRender, focusing on accurate representation of biological processes and experimental workflow.

---

## 🧪 Step 1: Selection of Research Content

A peer-reviewed research paper related to **biofilm development and analysis** was selected.  
The focus was on translating a **multi-step experimental workflow** into a visual format that highlights:

- Biofilm formation
- Biomass quantification
- Structural visualization
- Data interpretation

The protocol was carefully studied to identify key experimental stages and their sequence.

---

## 🖥️ Step 2: BioRender Setup

- Platform used: **BioRender (biorender.com)**
- A new figure was created using a **custom vertical layout (900 × 1200 px)** to accommodate multi-phase workflow representation.
- Background was kept **minimal (white)** to maintain clarity and readability.

---

## 🎨 Step 3: Structural Layout Design

The infographic was divided into **four distinct phases**, each represented as a horizontal block:

1. **Phase 1 – Biofilm Formation**
2. **Phase 2 – Biomass Quantification (Crystal Violet Assay)**
3. **Phase 3 – Advanced Visualization**
4. **Phase 4 – Data Analysis & Interpretation**

Each phase was arranged **top-to-bottom** to reflect logical experimental progression.

---

## 🧩 Step 4: Icon Selection and Placement

Relevant BioRender icons were selected to represent each experimental step:

- Cell culture inserts (Transwell / ALI-PBEC)
- Bacterial cells (mycobacteria)
- Pipettes and reagents
- 96-well plates
- Centrifuge and incubator
- Confocal and electron microscopes

Icons were arranged in a **left-to-right flow within each phase**, ensuring process continuity.

---

## 🔄 Step 5: Workflow Representation

- **Directional arrows** were added between steps to indicate progression.
- A **split-path layout** was used in Phase 3 to represent:
  - Confocal microscopy
  - SEM analysis

This improves clarity in representing parallel experimental techniques.

---

## 🎨 Step 6: Color Coding Strategy

To enhance readability, a **limited color palette (≤ 3 main colors)** was applied:

- 🟢 Teal → Biofilm formation  
- 🟣 Purple → Crystal violet assay  
- 🟠 Amber → Visualization techniques  
- 🔴 Coral → Data analysis  

This consistent color coding helps differentiate experimental stages.

---

## 🏷️ Step 7: Text and Labeling

- Title kept concise (**≤ 12 words**) and centered at the top.
- A short **one-line subtitle** was added to describe the workflow.
- Each step includes:
  - Process name  
  - Key conditions (e.g., MOI, time, concentration)  
  - Outcome  

Text was kept minimal but informative to avoid overcrowding.

---

## 📊 Step 8: Footer and Attribution

A clean footer was added for academic completeness:

- **Prepared by:** Vedanti Raut  
- **Source:** Barclay et al., 2025, *STAR Protocols*  

---

## 🖼️ Step 9: Export Settings

The final infographic was exported using:

- Format: **PNG**
- Resolution: **300 DPI**
- Additional check: **Grayscale preview** to ensure clarity without color dependency

---

## 📁 Step 10: Repository Integration

The exported infographic was uploaded to the GitHub repository with:

- Proper file naming  
- Organized folder structure (`/infographics/`)  
- This README is for documentation  

---

## ✅ Final Outcome

The final infographic:

- Accurately represents the research workflow  
- Uses structured visual hierarchy  
- Maintains clarity and scientific correctness  
- Follows BioRender and academic presentation standards
